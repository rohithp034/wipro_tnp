# Java Programming Exercises

This repository contains solutions to basic Java programming exercises covering topics such as:

- If-Else Statements
- Switch Statements
- For Loops
- While Loops
- Command Line Arguments
- Number Programs
- Pattern Programs

These programs are useful for beginners learning Java fundamentals.

---

## Requirements

- Java JDK 8 or above
- VS Code / IntelliJ IDEA / Eclipse (optional)

Check Java version:

```bash
java -version
javac -version
```

---

## Compile and Run

### Compile

```bash
javac FileName.java
```

Example:

```bash
javac P1A.java
```

### Run

```bash
java FileName
```

Example:

```bash
java P1A
```

For programs using command-line arguments:

```bash
java P3 Mumbai Bangalore
```
